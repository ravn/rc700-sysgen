ROA375 is the RC702 autoload rom.

Reports "** NO DISKETTE NOR LINEPROG"

The RC703 autoloader prom was found by Michael Ringgaard for his RC700 project (see https://www.jbox.dk/rc702/rcbios.shtm).
The source code for it is at https://www.jbox.dk/rc702/ROB358.MAC

I am recreating the source code for the ROA375 PROM using that as the inspiration.
